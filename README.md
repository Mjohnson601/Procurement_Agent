# C Spire Procurement Agent

An AI-powered procurement policy reference tool for C Spire employees.

## What It Does

Ask plain-language questions about C Spire procurement policy and get 
instant answers drawn from official policy documents:

- Procurement Policy (V1.0, effective May 1, 2026)
- Corporate Authorization Policy (CAP)
- Vendor Setup Policy
- Oracle Fusion Purchase Requisition Guide

## How to Use

Visit the live site or open `index.html` in a browser. No installation required.
